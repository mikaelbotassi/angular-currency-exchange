# QuoteApiService Documentation
___

The `QuoteApiService` is a service that allows you to fetch currency exchange rate data from an external API. This service provides a method for retrieving currency quotes.

## Usage

### Methods

#### `getQuotes(cod: string): Observable<Quote[]>`

This method sends an HTTP GET request to retrieve currency exchange rate data for a given currency code.

- `cod` (string): The currency code for which you want to fetch exchange rate data.

Returns an Observable that emits an array of `Quote` objects.

### Example

Here's an example of how to use the `QuoteApiService` to fetch currency exchange rate data:

```
import { Component, OnInit } from '@angular/core';
import { QuoteApiService } from './path-to-quote-api-service';
import { Quote } from './models/Quote';

@Component({
  selector: 'app-currency',
  templateUrl: './currency.component.html',
  styleUrls: ['./currency.component.css']
})
export class CurrencyComponent implements OnInit {
  quotes: Quote[] = [];

  constructor(private quoteApiService: QuoteApiService) { }

  ngOnInit(): void {
    // Fetch exchange rate data for the Argentine Peso (ARS)
    this.quoteApiService.getQuotes('ARS').subscribe((data: Quote[]) => {
      this.quotes = data;
    });
  }
}
```

In this example, we import the `QuoteApiService`, inject it into our component, and use the `getQuotes` method to fetch exchange rate data for the Argentine Peso (ARS).

That's it! You can now use the `QuoteApiService` in your Angular application to retrieve currency exchange rate data from the API.
