## Card Content Component

The `CardContentComponent` is used to display the content within a card, including currency information, variation percentage, and the last updated date.

### Usage

```
<app-card-content
  [isLoading]="isLoading"
  [error]="error"
  [textColor]="textColor"
  [currency]="currency"
  [percent]="percent"
  [created_date]="created_date"
  (restart)="sendReloadToParent()"
></app-card-content>
```

### Properties

- `isLoading` (Input): A boolean flag indicating whether the content is currently loading. If `true`, a loader will be displayed.

- `error` (Input): A boolean flag indicating whether an error has occurred. If `true`, an error message will be displayed.

- `textColor` (Input): A CSS class name that determines the text color for the currency value.

- `currency` (Input): The currency value to be displayed.

- `percent` (Input): The variation percentage to be displayed.

- `created_date` (Input): The date when the content was last updated.

- `restart` (Output): An event emitter that allows triggering a reload action when an error occurs.

### Change Detection Strategy

This component utilizes the `ChangeDetectionStrategy.OnPush` strategy to optimize performance by ensuring changes are detected only when component inputs change.
